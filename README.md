# Bank Account Console App (Java)

##  Overview

This is a **console-based banking application** built in Java that simulates basic account operations.
Users can deposit money, withdraw funds, check their balance, and view transaction history through an interactive menu.

This project was created to strengthen core programming skills and demonstrate clean program structure using methods and data structures.

---

##  Features

* 💰 Deposit money with input validation
* 💸 Withdraw money with balance checks
* 📊 View current account balance
* 🧾 Track and display transaction history
* 🔁 Continuous menu loop for user interaction
* 🚪 Exit option to safely close the program

---

## Concepts Demonstrated

* Java methods (modular programming)
* `while` loops and `do-while` loops
* Conditional logic (`if/else`, `switch`)
* User input handling with `Scanner`
* Dynamic data storage using `ArrayList`
* Input validation techniques
* Separation of concerns (main vs methods)

---

##  How It Works

The program runs in a loop and presents the user with a menu:

```
1. Deposit
2. Withdraw
3. Check Balance
4. View Transactions
5. Exit
```

* The user selects an option
* The program calls the appropriate method
* The balance and transaction history update accordingly

---

##  How to Run

1. Clone the repository:

   ```
   git clone https://github.com/richricketts13-ai/Bank-Account-V2.git
   ```

2. Navigate to the project folder:

   ```
   cd Bank-Account-V2
   ```

3. Compile the program:

   ```
   javac bankaccountmethods.java
   ```

4. Run the program:

   ```
   java bankaccountmethods
   ```
